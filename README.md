# Binary Search in Python

A simple Python implementation of the Binary Search algorithm with automated unit tests using pytest.

## Features

* Iterative Binary Search
* User input support
* Unit testing with pytest

## Project Structure

```text
binary_search_project/
├── src/
│   ├── binary_search.py
│   ├── main.py
│   └── __init__.py
├── tests/
│   └── test_binary_search.py
└── README.md
```

## Running the Program

```bash
python src/main.py
```

## Running Tests

```bash
python -m pytest
```

## Test Cases

The following scenarios are tested:

* Element found
* Element not found
* First element in the list
* Last element in the list
* Empty list

Expected result:

```text
5 passed
```
